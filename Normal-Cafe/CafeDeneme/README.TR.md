# Keyif Cafe — Web Sitesi

Samimi bir yerel kafe için yapılmış freelance proje. Sıcak krem ve kahverengi tonlar; menüyü sergilemek ve rezervasyonu kolaylaştırmak üzerine kurulu.

## ✨ Özellikler

- **Accordion menü** — her kategoriye tıklayınca Kahve, Çay ve Soğuk İçecekler açılıyor
- **Görsellli menü** — her içeceğin flat lay fotoğrafı, ismi, açıklaması ve fiyatı var
- **Google Maps** — gömülü konum haritası
- **WhatsApp rezervasyonu** — direkt WhatsApp iletişim butonu
- **Scroll animasyonları** — sayfayı kaydırdıkça bölümler beliriyor
- **Responsive tasarım** — tüm ekran boyutlarında çalışır
- **Sabit navbar** — blur efektli yapışkan navigasyon

## 🛠 Kullanılan Teknolojiler

- HTML5
- CSS3 (Grid, Flexbox, CSS geçişleri)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Jost)
- Yapay zeka ile üretilmiş içecek görselleri (Adobe Firefly)
