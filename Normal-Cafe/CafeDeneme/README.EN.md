# Keyif Cafe — Website

A freelance project built for a cozy local cafe. Warm cream and brown tones, focused on showcasing the menu and making reservations easy.

> _Coming soon / add link here_

## ✨ Features

- **Accordion menu** — click each category to expand Coffee, Tea, and Cold Drinks
- **Menu with photos** — each drink has a flat lay image, name, description and price
- **Google Maps** — embedded location map
- **WhatsApp reservation** — direct WhatsApp contact button
- **Scroll animations** — sections fade in as you scroll
- **Responsive design** — works on all screen sizes
- **Sticky navbar** — fixed navigation with blur backdrop

## 🛠 Built With

- HTML5
- CSS3 (Grid, Flexbox, CSS transitions)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Jost)
- AI-generated drink images (Adobe Firefly)
