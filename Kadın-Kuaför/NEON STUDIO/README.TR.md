# NEON STUDIO — Premium Saç Sanatı Sitesi

Bir kadın kuaför salonu için yapılmış freelance proje. Neon cyan aksanlı fütüristik koyu tema; imza saç modellerini sergilemek ve randevu almayı kolaylaştırmak üzerine kurulu.

## ✨ Özellikler

- **İnteraktif saç kataloğu** — stil kartlarının üzerine gelince önden, yandan ve arkadan görünümler arasında geçiş yapar
- **Scroll animasyonları** — sayfayı kaydırdıkça bölümler fade-in ile belirir
- **Randevu formu** — hizmet seçimi ve tarih seçici içeren randevu talebi formu
- **Hizmetler bölümü** — hover efektli 4 hizmet kartı
- **Müşteri yorumları** — referans bölümü
- **Responsive tasarım** — tüm ekran boyutlarında çalışır
- **Sabit navbar** — blur efektli yapışkan navigasyon

## 🛠 Kullanılan Teknolojiler

- HTML5
- CSS3 (Grid, Flexbox, CSS geçişleri)
- Vanilla JavaScript
- Google Fonts (Syne, Inter)
- Yapay zeka ile üretilmiş saç modeli görselleri (Leonardo AI)
