# NEON STUDIO — Premium Hair Artistry Website

A freelance project built for a women's hair salon. Futuristic dark theme with neon cyan accents, focused on showcasing signature hairstyles and booking appointments.

## 🔗 Live Demo

https://neon-hair-studio.netlify.app

## ✨ Features

- **Interactive hair catalog** — hover over style cards to cycle through front, side and back angles
- **Scroll animations** — sections fade in as you scroll down the page
- **Booking form** — appointment request form with service selection and date picker
- **Services section** — 4 service cards with hover effects
- **Testimonials** — client review section
- **Responsive design** — works on all screen sizes
- **Sticky navbar** — fixed navigation with blur backdrop

## 🛠 Built With

- HTML5
- CSS3 (Grid, Flexbox, CSS transitions)
- Vanilla JavaScript
- Google Fonts (Syne, Inter)
- AI-generated hairstyle images (Leonardo AI)

## 📸 Design

Dark futuristic theme with near-black (`#050505`) background and neon cyan (`#00f2ff`) accents. Typography uses Syne for headings and Inter for body text.
