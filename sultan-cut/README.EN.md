# SULTAN CUT — Premium Barber Shop Website

A freelance project built for a premium barber shop client. Clean, dark luxury design with gold accents, focused on showcasing services and driving appointment bookings.

## ✨ Features

- **Responsive design** — works on all screen sizes
- **Interactive gallery** — hover over hairstyle cards to cycle through front, side and back angles
- **Pricing packages** — Classic, Premium and Royal tiers with featured card highlight
- **Services section** — 6 service cards with hover animations
- **Contact form** — appointment booking form with focus states
- **Smooth scroll** — navigation links scroll smoothly to each section
- **Sticky navbar** — fixed header with blur backdrop

## 🛠 Built With

- HTML5
- CSS3 (Grid, Flexbox, custom animations)
- Vanilla JavaScript
- Google Fonts (Playfair Display, DM Sans)
- AI-generated hairstyle images (Leonardo AI)

## 📸 Design

Dark luxury theme with black (`#0e0e0e`) background and gold (`#c9a84c`) accents. Typography uses Playfair Display for headings and DM Sans for body text.
